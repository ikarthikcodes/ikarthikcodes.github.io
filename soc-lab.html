SOC Detection Lab

Brute-force detection, log correlation & MITRE ATT&CK mapping

Project Overview

Built a home SOC lab to simulate brute-force SSH attacks and detect them using Wazuh and Splunk. The objective was to analyze logs, create alert rules, and map attack techniques to MITRE ATT&CK.

Tools Used: Wazuh • Splunk • Ubuntu • Kali Linux • MITRE ATT&CK
View on GitHub

Lab Architecture

The lab consisted of a Kali attacker machine and a vulnerable Ubuntu server. Logs were forwarded to Wazuh manager and visualized in Splunk for analysis.

Attack Simulation

Simulated SSH brute-force attack using Hydra from Kali Linux. Observed failed login attempts and analyzed authentication logs.

Detection & MITRE Mapping

Created custom alert rules to detect repeated failed SSH attempts. Mapped activity to:

  • T1110 — Brute Force
  • T1021 — Remote Services
